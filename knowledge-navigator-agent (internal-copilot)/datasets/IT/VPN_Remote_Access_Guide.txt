# VPN & Remote Access Guide

**Version 2.8 | IT Department**

## What is VPN?

Virtual Private Network (VPN) creates secure, encrypted connection between your device and company network. Required for accessing company resources from outside the office.

## When VPN is Required

### Always Use VPN When:
- Working from home or remote location
- Using public WiFi (coffee shop, airport, hotel)
- Accessing internal systems or applications
- Viewing confidential company data
- Using company email on laptop (auto-required)
- Connecting to file shares or databases

### VPN Not Required:
- In the office on wired network
- Using cloud applications with web login (Salesforce, etc.)
- Accessing public company website
- Using company WiFi in office

## Setting Up VPN

### Installation

**Windows:**
1. Go to helpdesk.company.com/vpn
2. Download "Cisco AnyConnect" installer
3. Run installer (admin rights auto-granted)
4. Follow on-screen prompts
5. Restart computer

**Mac:**
1. Visit helpdesk.company.com/vpn
2. Download "Cisco AnyConnect" for Mac
3. Open .dmg file and run installer
4. Allow security permissions when prompted
5. Restart computer

**Mobile (iOS/Android):**
1. Download "Cisco AnyConnect" from App Store/Play Store
2. Open app and tap "Add VPN Connection"
3. Server: vpn.company.com
4. Username: Your company email
5. Password: Your network password

### Initial Configuration
1. Open AnyConnect application
2. Server address auto-populated: vpn.company.com
3. Enter username (company email address)
4. Enter password (same as computer login)
5. Enter MFA code from Authenticator app
6. Click "Connect"
7. Save credentials for faster login (optional)

## Using VPN

### Connecting to VPN
1. Open Cisco AnyConnect
2. Toggle switch to "Connect" or click "Connect" button
3. Enter password and MFA code (if not saved)
4. Wait for "Connected" status (10-30 seconds)
5. Green checkmark indicates successful connection

### Status Indicators
- **Green Connected:** VPN active, secure connection
- **Yellow Connecting:** In process, wait a moment
- **Red Disconnected:** Not connected or connection failed
- **Orange Reconnecting:** Trying to restore connection

### Auto-Connect
- VPN auto-connects when accessing certain apps
- Can enable "Connect on Demand" in settings
- Reconnects automatically if dropped
- Stays connected until you disconnect or shut down

### Disconnecting
1. Open AnyConnect
2. Click "Disconnect" button
3. Confirm disconnection
4. Safe to close application

**When to Disconnect:**
- Done with work for the day
- Switching to personal internet usage
- Experiencing slow connection
- Troubleshooting network issues

## Troubleshooting

### Cannot Connect to VPN

**Error: "Login Failed" or "Authentication Failed"**
- Verify username is correct (full email address)
- Check password (same as computer login)
- Ensure MFA code is current (codes expire every 30 seconds)
- Check if account is locked (too many failed attempts)
- Wait 30 minutes if locked, then try again
- Contact IT if still failing: (555) 123-4000

**Error: "Unable to Contact VPN Server"**
- Check internet connection (open browser, visit google.com)
- Verify server address: vpn.company.com
- Try disconnecting and reconnecting
- Restart computer
- Check if VPN service is down: status.company.com
- Contact IT if issue persists

**Error: "Certificate Error" or "Untrusted Connection"**
- Certificate may have expired
- Uninstall AnyConnect completely
- Download fresh copy from helpdesk.company.com/vpn
- Reinstall and try again
- Contact IT if error continues

### VPN Connected But Can't Access Resources

**Can't Access File Shares or Applications:**
- Confirm VPN shows "Connected" (green)
- Try accessing different resource
- Disconnect and reconnect VPN
- Flush DNS: Open Command Prompt, type `ipconfig /flushdns` (Windows) or `sudo dscacheutil -flushcache` (Mac)
- Restart computer
- Contact IT with specific error messages

**VPN is Very Slow:**
- Check internet speed (speedtest.net) - need minimum 10 Mbps
- Disconnect from VPN, reconnect
- Close unnecessary applications
- Use VPN split tunneling (IT can enable)
- Try different WiFi network
- Use phone hotspot if WiFi is slow
- Peak usage times (9-11 AM) may be slower

### Other Issues

**VPN Keeps Disconnecting:**
- Check WiFi signal strength
- Update AnyConnect to latest version
- Disable power saving on network adapter (Windows)
- Change WiFi channel on router (if home network)
- Try wired connection instead of WiFi
- Contact IT if frequent disconnections

**MFA Not Working:**
- Ensure time on phone is set to automatic
- Try backup MFA method (SMS or phone call)
- Generate new code after 30 seconds
- Contact IT to reset MFA: (555) 123-4100

**AnyConnect Won't Launch:**
- Restart computer
- Check if AnyConnect service is running
- Uninstall and reinstall AnyConnect
- Contact IT for assistance

## Security Best Practices

### Protect Your Connection
- Always use VPN on public WiFi
- Don't share VPN credentials
- Log out when finished working
- Don't let others use your VPN connection
- Report lost devices immediately

### Public Networks
- Verify network name with establishment
- Beware of fake hotspots ("Free WiFi", "Guest Network")
- Don't enter sensitive data before connecting VPN
- Use phone hotspot for most secure option
- Disable auto-connect to open networks

### MFA Security
- Don't share MFA codes with anyone
- IT will never ask for your MFA code
- Approve MFA prompts you initiated only
- Report unexpected MFA prompts to security
- Keep authenticator app on your phone only

### Web-Based Access

## Remote Desktop Access

### When to Use Remote Desktop
- Need full access to office computer
- Software only available on office machine
- Accessing file on office computer hard drive
- Testing applications in office environment

### Connecting via Remote Desktop
1. Connect to VPN first
2. Open "Remote Desktop" app
3. Enter computer name: DESKTOP-XXXXX (provided by IT)
4. Enter credentials
5. Access your office computer remotely

### Remote Desktop Best Practices
- Close sensitive files before leaving office
- Lock office computer (don't shut down)
- Disconnect when done to free resources
- Don't leave remote session open overnight
- Report issues: helpdesk@company.com

## Performance Optimization

### Improve VPN Speed
- Close unnecessary applications and browser tabs
- Disconnect from VPN when not actively working
- Use split tunneling (IT can enable per request)
- Connect via wired ethernet instead of WiFi
- Upgrade home internet if consistently slow
- Work during off-peak hours when possible

### Reduce Bandwidth Usage
- Don't stream video/music while on VPN
- Download large files directly (not through VPN)
- Use web versions of apps when possible
- Disable automatic cloud sync while on VPN
- Compress files before uploading

## FAQs

**Q: Do I need VPN if I'm using my home WiFi?**
A: Yes, VPN required when working remotely to securely access company resources and protect data.

**Q: Can I use VPN for personal internet browsing?**
A: You can, but it may slow down your connection. Disconnect VPN for personal use, reconnect for work.

**Q: Why does VPN disconnect when my computer sleeps?**
A: Normal behavior. VPN reconnects automatically when computer wakes, or you can reconnect manually.

**Q: Can I use VPN on multiple devices simultaneously?**
A: Yes, you can be connected on phone and laptop at same time with same credentials.

**Q: Will VPN slow down my internet?**
A: Slightly, due to encryption. With good internet (25+ Mbps), impact is minimal.

**Q: What if I'm traveling internationally?**
A: VPN works worldwide. May need to try multiple times in countries with internet restrictions. Contact IT before travel to restricted countries.

**Q: Can IT see what I'm doing on VPN?**
A: IT can see you're connected and which company resources you access. Web browsing through VPN may be logged per security policy.

**Q: What if I forgot my password?**
A: Reset at password.company.com or call IT: (555) 123-4000. VPN password is same as your computer login.
